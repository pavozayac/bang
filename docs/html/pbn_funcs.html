class PBN¶

Class representing Probabilistic Boolean Networks.

This class stores information about a Probabilistic Boolean Networks, parses informations about PBNs from files and formats data for simulation.

Variables:

  • n (int) – number of nodes.

  • nf (list[int]) – number of boolean functions for each node.

  • nv (list[int]) – number of variables for each boolean function.

  • F (list[list[bool]]) – truth table of each boolean function.

  • varFInt (list[list[int]]) – number of node each variable of boolean function references

  • cij (list[list[float]]) – probability of each boolean function to be picked.

  • perturbation (float) – perturbation rate.

  • npNode (list[int]) – index of nodes without perturbation

load_from_file(path, format='sbml') → PBN¶

Loads file of a given format from path and creates object of class PBN from it. (sbml files are still WIP)

Parameters:

  • format (str) – format of the file. Can be ‘sbml’ for .sbml files or ‘assa’ for .pbn files. Devaults to ‘.sbml’
